
# Simple Todo App

## Description
The Simple Todo App is a lightweight and user-friendly application that helps users manage their daily tasks efficiently. With a clean interface and essential features, it is perfect for keeping track of your to-do list.

## Features
- Add new tasks to your to-do list.****
- Delete tasks that are no longer needed.


## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project *irectory**:
   ```bash
   cd Simple-Todo-App
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
